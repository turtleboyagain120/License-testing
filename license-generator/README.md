# License Generator

## Quick Start
1. `cd license-generator`
2. `pip install -r requirements.txt` (installs Flask)
3. `python backend.py` (starts server at http://localhost:5000)
4. Open `index.html` in browser (or visit http://localhost:5000/index.html)

Fill 10 questions → Generate & download custom LICENSE.md based on your templates.

## Features
- 10-question questionnaire (project type, risk, restrictions)
- Scores/matches your 20+ personal licenses
- Customizes placeholders (name, year, etc.)
- Local-only, no internet

Built with HTML/JS/CSS frontend, Flask Python backend, JSON data.
